# 虞美人·春花秋月何时了 #  
### 五代：李煜 ###
------------
> 这是引用 *李煜* 的词  

* 春花秋月何时了？
* 往事知多少。
* 小楼昨夜又东风，
* 故国不堪回首月明中。
* 雕栏玉砌应犹在，
* 只是朱颜改。
* 问君能有**几多愁**？
* 恰似一江春水向东流。
